## Présentation

PopCode est un jeu inspiré de [Pop Corn Garage]("https://popcorngarage.com/), un jeu de cherche et trouve des titres de films. Le but dans PopCode est de trouver le nom des langages de programmation.  
Il est possible de zoomer dans l'image pour mieux voir les détails.

Le jeu n'est pas malheureusement pas <span className='text-[8px]'>"encore"</span> responsive, il est préférable d'y jouer sur un ordinateur.

## Les règles

- Trouver le nom des langages de programmation
- 27 langages à trouver
- Taper votre réponse à l'aide du clavier
- Vous avez 3 essais, au delà de 3 erreurs, vous perdez la partie
